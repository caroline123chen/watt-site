Drop-in integration

Point your base URL at Watt. Nothing else changes.

pythonnodecurl
# pip install openai — same SDK, one base URL
from openai import OpenAI

client = OpenAI(
    base_url="https://api.watt.computenotes.co/v1",
    api_key="watt-...",
)

resp = client.chat.completions.create(
    model="claude-sonnet-4-6",   # or gpt-4o, gemini, deepseek-chat ...
    messages=[{"role": "user", "content": "Hello, Watt"}],
    stream=True,
)

FAQ

Questions, answered.

Is Watt OpenAI-compatible?

Yes. Change one base URL and your existing OpenAI SDK code works unchanged — streaming, tool calls and structured outputs included.

Which models can I call?

Claude, GPT, Gemini, DeepSeek, MiniMax, GLM, Llama, Qwen and more — all through one key. New models are added regularly.

How does billing work?

Simple AUD subscriptions via Stripe, with proper GST tax invoices. Usage draws against your plan; you see spend and cost-by-model in the dashboard.

Is my data used for training?

No. Your prompts and outputs are never used for training. Watt logs only metadata — model, tokens and cost.

Where is Watt based?

Australia. AUD billing, GST invoices, local timezone support, and a real person behind the account.
